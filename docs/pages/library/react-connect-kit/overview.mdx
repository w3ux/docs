# React Connect Kit

Providers and hooks for connecting to web3 wallets and interacting with accounts.

[npm](https://www.npmjs.com/package/@w3ux/react-connect-kit) |&nbsp; [source](https://github.com/w3ux/w3ux-library/tree/main/library/react-connect-kit)

## Install

First, we will install `@w3ux/react-connect-kit` as a dependency in our project.

:::code-group

```bash [npm]
npm i @w3ux/react-connect-kit
```

```bash [pnpm]
pnpm i @w3ux/react-connect-kit
```

```bash [yarn]
yarn add @w3ux/react-connect-kit
```

```bash [bun]
bun i @w3ux/react-connect-kit
```

:::
