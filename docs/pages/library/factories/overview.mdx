# Factories

A collection of general purpose TypeScript factories.

[npm](https://www.npmjs.com/package/@w3ux/factories) |&nbsp; [source](https://github.com/w3ux/w3ux-library/tree/main/library/factories)

## Install

First, we will install `@w3ux/factories` as a dependency in our project.

:::code-group

```bash [npm]
npm i @w3ux/factories
```

```bash [pnpm]
pnpm i @w3ux/factories
```

```bash [yarn]
yarn add @w3ux/factories
```

```bash [bun]
bun i @w3ux/factories
```

:::
