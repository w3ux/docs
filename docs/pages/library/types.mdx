# Types

Common types used throughout the w3ux library.

[npm](https://www.npmjs.com/package/@w3ux/types) |&nbsp; [source](https://github.com/w3ux/w3ux-library/tree/main/library/types)


::::steps

### Install

First, we will install `@w3ux/types` as a dependency in our project.

:::code-group

```bash [npm]
npm i @w3ux/types
```

```bash [pnpm]
pnpm i @w3ux/types
```

```bash [yarn]
yarn add @w3ux/types
```

```bash [bun]
bun i @w3ux/types
```

:::

### Import

```tsx
import { Sync } from '@w3ux/types';

let synced: Sync = 'unsynced';
```
