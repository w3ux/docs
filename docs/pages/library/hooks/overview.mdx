# Hooks

A collection of generic React hooks.

[npm](https://www.npmjs.com/package/@w3ux/hooks) |&nbsp; [source](https://github.com/w3ux/w3ux-library/tree/main/library/hooks)

## Install

First, we will install `@w3ux/hooks` as a dependency in our project.

:::code-group

```bash [npm]
npm i @w3ux/hooks
```

```bash [pnpm]
pnpm i @w3ux/hooks
```

```bash [yarn]
yarn add @w3ux/hooks
```

```bash [bun]
bun i @w3ux/hooks
```

:::
